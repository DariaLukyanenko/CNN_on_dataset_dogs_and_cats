# CNN_on_dataset_dogs_and_cats

# Cats vs Dogs Classification with PyTorch

## Описание
Блокнот посвящён задаче бинарной классификации изображений кошек и собак с помощью PyTorch.  
В работе последовательно разбираются подготовка датасета, обучение базовой полносвязной сети и переход к сверточной архитектуре.

## Данные
Используется уменьшенная версия датасета **Dogs vs Cats**.

Структура данных:
- `train_11k` — обучающая выборка;
- `val` — валидационная выборка;
- `test_labeled` — тестовая выборка.

Подготовка данных включает:
- загрузку через `ImageFolder`;
- приведение изображений к размеру `96x96`;
- преобразование в тензоры;
- нормализацию по каналам.

## Что сделано в блокноте
- настройка `Dataset` и `DataLoader`;
- визуализация изображений;
- реализация общего цикла обучения и валидации;
- оценка качества на validation и test;
- обучение baseline-модели на полносвязных слоях;
- улучшение MLP через инициализацию Xavier, dropout, weight decay и gradient clipping;
- переход к сверточной сети;
- использование `BatchNorm2d`, `Dropout2d` и `MaxPool2d`;
- просмотр предсказаний модели на изображениях.

## Архитектуры

### 1. Базовая MLP
Полносвязная сеть после `Flatten`:
- `Linear(27648 -> 256)`;
- `ReLU`;
- `Dropout`;
- `Linear(256 -> 128)`;
- `ReLU`;
- `Dropout`;
- выходной слой на 2 класса.

### 2. Улучшенная MLP
Добавлены:
- более аккуратная инициализация весов;
- увеличенный dropout;
- L2-регуляризация через `weight_decay`;
- ограничение нормы градиента.

### 3. CNN
Сверточная сеть с несколькими блоками:
- `Conv2d`;
- `ReLU`;
- `MaxPool2d`;
- `Dropout2d`;
- `BatchNorm2d`;
- полносвязная классификационная голова.

## Обучение
Используются:
- `CrossEntropyLoss`;
- оптимизатор `Adam`;
- обучение на `CPU/GPU`;
- валидация после каждой эпохи.

## Результаты
### Базовая полносвязная сеть
- validation accuracy: около **65.11%**
- test accuracy: около **63.94%**

### Улучшенная MLP
- validation accuracy остаётся около **64–65%**
- заметного прироста по сравнению с baseline нет

### CNN
- лучшая validation accuracy: около **88.24%**
- итоговая validation accuracy: **87.15%**
- test accuracy: **86.53%**

## Вывод
Полносвязная сеть плохо справляется с задачей классификации реальных изображений кошек и собак.  
Переход к сверточной архитектуре даёт существенный прирост качества, так как CNN лучше извлекает пространственные признаки изображения.  
Основной результат блокнота — показать разницу между MLP и CNN на реальной задаче компьютерного зрения.
